README for Mastermind Networked Game Application
=================================================

Setup Instructions:
-------------------
1. Run 'ServerMain.java' to start the server.
2. Run 'ClientMain.java' to start a client.
3. Repeat step 2 for multiple clients.

Gameplay:
---------
- Guess the server's secret code to win.
- Each incorrect guess receives feedback.
- Game ends when the code is guessed or all guesses are used.
